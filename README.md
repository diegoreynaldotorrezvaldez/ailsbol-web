# AILSBOL WEB

Sistema web integral de gestión y formación para la Asociación de Intérpretes de Lengua de Señas de Bolivia (AILSBOL).

## Primer Sprint

El primer sprint implementa la base funcional del sistema:

- Página pública institucional.
- Inicio de sesión.
- Panel administrativo.
- Dashboard con datos reales.
- Gestión básica de usuarios.
- Visualización de roles.
- Registro de usuarios.
- Activación y desactivación de usuarios.
- Conexión con base de datos MySQL.

## Tecnologías utilizadas

- Laravel 9
- React.js con Vite
- MySQL
- Axios
- React Router
- XAMPP para entorno local
